The Fleet That Never Sleeps

Instead of relying on a single AI, this project uses two specialised agents working in parallel on the same codebase. Each agent has a defined role, its own memory file, and picks up tickets from a shared Kanban board. The human acts as Switchboard Manager — reviewing PRs and keeping the sprint on track.

The Application

A fully functional CRM with contacts, tasks, activity log, and a task pipeline — built from scratch in a single afternoon session.

1
Home Dashboard
Two-column view: Recent Activities on the left, Active Tasks on the right. Updates in real time as contacts are edited or tasks are completed.
Recent ActivitiesAuto-logged when any contact is updated — shows field-level diff (e.g. status: lead → active)
Active TasksAll open tasks with contact link, category, due date and assignee
+ Add TaskGlobal quick-add from any screen
2
Contacts List
Sortable, searchable table of all contacts with live search hitting the API.
SortClick Name or Company header to sort ascending / descending
SearchSearches name, company, email and location simultaneously
FilterFilter by status: Active, Lead, Inactive
TagsColour-coded tag pills per contact
3
Contact Detail
Click any contact row to open their detail view — tasks, info, and quick actions in one place.
Contact TasksAll tasks linked to this specific contact
Log ActivityRecord a call, meeting or note against the contact
Edit ContactUpdate any field including status via dropdown
4
Edit Contact
Full edit form with status dropdown. Every save automatically writes an activity entry showing exactly what changed.
Status DropdownActive / Lead / Inactive — one click to update pipeline stage
Auto Activity LogBackend diffs old vs new and writes e.g. "Updated: status: 'lead' → 'active'"
ValidationRequired fields enforced with inline error messages
5
Add Task Modal
Global task creation — or launched from a Contact Detail to pre-link the contact automatically.
CategoryCall, Meeting, Follow-up, Email, Demo, Other
Due DateDate picker with required validation
Linked ContactPre-filled and locked when opened from Contact Detail
AssigneeLink task to a team member
6
Task Kanban
Visual pipeline board — drag tasks across To Do / In Progress / Closed with one click.
Three stagesTo Do → In Progress → Closed
One-click moveButtons on each card to advance or revert stage
Contact linkEach card shows the linked contact name
